## 🌾 Overview

**Agrirent** is a modern, responsive web application built with React.js for discovering, browsing, and renting agricultural equipment. Farmers and renters can explore tractors, harvesters, implements, and more with detailed listings, user authentication, and intuitive navigation.

Perfect for agritech enthusiasts, this mini-project demonstrates full-stack frontend capabilities with routing, API integration (via Axios), and Bootstrap styling.

![Agrirent Screenshot](screenshots/home.png) <!-- Add screenshots here -->

## ✨ Features

- **Home Dashboard**: Welcome page with featured equipment.
- **Equipment Catalog**: Browse all available rentals with search/filter.
- **Equipment Details**: In-depth views with specs, pricing, images.
- **User Authentication**: Login & Register functionality.
- **About & Contact**: Static pages for business info.
- **Responsive Design**: Mobile-first with Bootstrap.
- **React Router**: Seamless single-page navigation.

## 🛠 Tech Stack

- **Frontend**: React 19, React Router DOM 6, React Bootstrap
- **Styling**: Bootstrap 5, Custom CSS
- **HTTP**: Axios
- **Build**: Create React App
- **Testing**: React Testing Library

## 🚀 Quick Start

1. **Clone the repo**:
   ```
   git clone https://github.com/padvijatin/agrirent-react-mini-project.git
   cd agrirent-react-mini-project
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run development server**:
   ```
   npm start
   ```
   Opens at [http://localhost:3000](http://localhost:3000)

4. **Build for production**:
   ```
   npm run build
   ```

## 🌐 Deployment

- **Vercel**: Connect GitHub repo, auto-deploys on push.
- **Netlify**: Drag `build/` folder or Git integration.
